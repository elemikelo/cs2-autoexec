# 🔥 CS2 Autoexec & Movement Config

Configuración avanzada para **Counter‑Strike 2** orientada a:

-   Movement
-   KZ
-   Long Jump
-   Count Jump
-   Practice

Incluye una **configuración modular profesional** con instalación
automática mediante script.

------------------------------------------------------------------------

# 🚀 Características

## 🎮 Movement

Configuración optimizada para movement:

-   Bhop optimizado
-   Scripts para **Long Jump**
-   Scripts para **Count Jump**
-   Compatible con servidores **KZ**

------------------------------------------------------------------------

## ⚙️ Configuración modular

La configuración está dividida en módulos independientes para mantener
el proyecto limpio y fácil de mantener.

    core
    crosshair
    viewmodel
    radar
    audio
    movement
    binds
    utilities

Esto permite modificar partes específicas de la configuración sin
afectar al resto.

------------------------------------------------------------------------

## 📸 Photo Mode

Modo especial para capturas o clips.

Incluye:

-   HUD oculto
-   Ajuste de gamma
-   Desenfoque de fondo
-   Cambio de viewmodel

Comando:

    photo

------------------------------------------------------------------------

## 🔇 Clutch Mute

Mutea automáticamente el voice chat para concentrarte en clutch.

Tecla:

    Right ALT

------------------------------------------------------------------------

## 💣 C4 Quick Drop

Drop rápido de C4 para pasarla a compañeros.

Tecla:

    J

------------------------------------------------------------------------

## 🔊 Control rápido de volumen

  Tecla   Volumen
  ------- ---------
  F4      Bajo
  F5      Medio
  F6      Alto

# ⚡ Instalación automática (recomendado)

1️⃣ Descarga el repositorio

    git clone https://github.com/TU_USUARIO/cs2-autoexec

o descarga el **ZIP**.

2️⃣ Ejecuta:

    install_cfg.bat

El instalador:

-   detecta automáticamente la carpeta de configuración de CS2
-   copia todos los archivos `.cfg` del proyecto
-   instala la configuración completa automáticamente

------------------------------------------------------------------------

# ⚙️ Activar autoexec en Steam

1.  Abre **Steam**
2.  Ve a tu biblioteca
3.  Click derecho en **Counter‑Strike 2**
4.  Propiedades
5.  En **Opciones de lanzamiento** añade:

```{=html}
<!-- -->
```
    +exec autoexec.cfg

------------------------------------------------------------------------
